2. Welche Daten wir verarbeiten

Beim Nutzen des HU-Vorab-Checks verarbeiten wir folgende Daten:

  • Fotos deines Fahrzeugs — zur KI-Analyse, werden nicht dauerhaft gespeichert
  • E-Mail-Adresse — für die Zahlungsabwicklung via Stripe
  • Kennzeichen (optional) — wenn du es angibst, zur Verbesserung der Analyse
  • Technische Daten — IP-Adresse, Browser, Zugriffszeit (Server-Logs)

3. KI-Verarbeitung (Anthropic Claude)

Die Foto-Analyse erfolgt durch Claude (Anthropic). Deine Fotos werden zur Analyse an die Anthropic API übermittelt und dort verarbeitet. Anthropic speichert die Daten gemäß ihrer Datenschutzrichtlinie (anthropic.com/privacy). Die Fotos werden von uns nicht dauerhaft gespeichert.

4. Zahlungsabwicklung (Stripe)

Zahlungen werden über Stripe Inc. abgewickelt. Dabei werden deine Zahlungsdaten direkt an Stripe übertragen und dort verarbeitet. Wir speichern keine Kreditkartendaten. Weitere Informationen: stripe.com/de/privacy

5. Hosting (Supabase & GitHub Pages)

Diese Seite wird über GitHub Pages bereitgestellt. Backend-Funktionen laufen über Supabase (Server in der EU). Beide Anbieter verarbeiten technische Zugriffsdaten gemäß ihrer Datenschutzrichtlinien.
